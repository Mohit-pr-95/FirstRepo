# Amazon Clone

A responsive *Amazon India homepage clone* built using *HTML5* and *CSS3*. This project recreates the overall look and feel of Amazon's mobile website, including the navigation bar, search section, product categories, promotional banners, shopping cards, and user interface elements.

*Note: This project is created purely for learning and educational purposes. It is not affiliated with or endorsed by Amazon.*

# Features:

• Responsive mobile-inspired layout
• Amazon-style navigation bar
• Search bar with search button
• Cart and profile icons
• Product category cards
• Promotional banners
• Multiple shopping sections
• Deals and offer cards
• Sticky navigation section
• Multiple HTML pages for navigation
• Clean UI built entirely with HTML & CSS
• Custom SVG icons

# Technologies Used:

• HTML5
• CSS3
